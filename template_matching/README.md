## 1.开发环境说明
 - 基于python3.和opencv3.开发，需要安装python的一些库，numpy,matplotlib//或者Anaconda
- 基于模板匹配的手写数字识别，采用mnist部分数据集
-----
## 2.原理
>基于模板匹配的数字识别，将标准的28*28像素的数字0~9读取，二值化，对每个数字进行等分区域分割，统计每个区域内的黑色像素点的个数，即为特征初值.采用欧式距离的模板匹配法与训练得到的特征值进行比较。
