## 2.[原理](\实验原理.JPG)
>2.1 基于模板匹配的数字识别，将标准的28*28像素的数字0~9读取，二值化，对每个数字进行等分区域分割压缩7**7，统计每个区域内的白色像素点的个数，即为特征初值.将得到的特征值字符串化0100

>2.2 计算先验概率
     p(wi)= Ni/N  
 每类样本数除以每类样本总数

>2.3 计算条件概率pj(wi)，Ni张样本每列特征值中为1的概率

>2.4 计算类条件概率 

        P（X|Wi）= P(X=a|Wi) a=0,1
